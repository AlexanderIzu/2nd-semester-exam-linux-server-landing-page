About the Project

Project Setup and Deployment Process

  • Provisioned a Linux server using AWS EC2 with Ubuntu.
  • Installed a web server (Apache) to serve web content.
  • Deployed this simple HTML page to the server.
  • Configured HTTP traffic (port 80) for public access.
  • Configured HTTPS with Let’s Encrypt using Certbot
  • Created a Repository and pushed the necessary files to GitHub
